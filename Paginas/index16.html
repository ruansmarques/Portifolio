Sistema de caixas

Composição de uma caixa

Todo elemento possui uma:

  • Altura: "height"
  • Largura: "width"

    • Ao criar uma borda em um elemento, o seu contorno ocupa o limite da borda da caixa do elemento. Para criar uma borda, basta usar a tag:

  • "border"

    • Para afastar a borda do limite da caixa, bastar usar a tag:

  • "padding"

    • Para afastar qualquer outro elemento do limite de uma borda, basta usar a tag:

  • "margin"

    • É possivel criar uma outra borda além de uma já criada, deixando as duas juntas uma da outra. Para isso, basta usar a tag:

  • "outline"

    • Tipos de caixas

    Há dois tipos de caixas, sendo o tipos "box-level" e do tipo "in-line". Uma caixa "box-level" possui uma quebra de linha automática após sua adição. Já uma caixa "in-line" não possui quebra de linha, caso necessário pode-se aplicar uma pela tag "hr".

    Box-level

    Alguns elementos do tipo "box-level" são:

  • "div"
  • "h1-h6"
  • "p"
  • "main"
  • "header"
  • "nav"
  • "article"
  • "aside"
  • "footer"
  • "form"
  • "video"

    • Inline-level

    Alguns elementos do tipo "in-line" são:

  • "span"
  • "a"
  • "code"
  • "small"
  • "strong"
  • "em"
  • "sup-sub"
  • "label"
  • "button"
  • "input"
  • "select"

    • Shorthands

    Shorthands são atalhos para simplificar a transcrição de um código, onde é retirado a necessidade de colocar as propriedades de uma estilização, colocando apenas os seus parâmentros.

    Bordas

    Logo, troca-se o parâmentro "1." por "2."

    1. border-width: 10px;
      border-style: solid;
      color: blue;

    2. border: 10px solid blue;

    Contorno

    Logo, troca-se o parâmentro "1." por "2."

    1. outline-width: 10px;
      outline-style: solid;
      color: blue;

    2. outline: 10px solid blue;

    Preenchimento

    Logo, troca-se o parâmentro "1." por "2."

    1. padding-top: 10px;
      padding-right: 10px;
      padding-bottom: 10px;
      padding-left: 10px;

    2. padding: 10px 10px 10px 10px;

    Margem

    Logo, troca-se o parâmentro "1." por "2."

    1. margem-top: 10px;
      margem-right: 10px;
      margem-bottom: 10px;
      margem-left: 10px;

    2. margem: 10px 10px 10px 10px;

    Tipos de Grouping Tags

    Servem para organizar as etapas de transcrição do código de um site. No protocolo HTML4, as Grouping Tags eram organizadas por diversas "divs" identificadas por um seletor "id". Atualmente, o HTML5 possui tags específicas para cada etapa da programação.

    Cabeçalho

    Usada para fixar informações principais de um site. Para aplicá-lo, usa-se a tag:

  • "header"

    • Navegação

    Usada como uma sub-divisão do cabeçalho. Para aplicá-lo, usa-se a tag:

  • "nav"

    • Principal

    Usada para receber o conteúdo principal do site. Para aplicá-lo, usa-se a tag:

  • "main"

    • Sessão

    Usada como uma sub-divisão do Principal. Para aplicá-lo, usa-se a tag:

  • "section"

    • Artigo

    Usada como uma sub-divisão da Sessão. Para aplicá-lo, usa-se a tag:

  • "article"

    • À parte

    Usada como uma sub-divisão dos Artigos. Para aplicá-lo, usa-se a tag:

  • aside

    • Rodapé

    Usada para fixar informações complementares sobre o site. Para aplicá-lo, usa-se a tag:

  • "footer"
